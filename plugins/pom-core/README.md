# pom-core

The core plugin for the [Product Operating Model](../../README.md) marketplace.

> Placeholder — full plugin README ships in Phase C of the v0.1.0 build.

## What's inside (v0.1.0 target)

- **15 skills** (`pom-*`) for intake, scoring, gating, routing, promotion, runway, validation, status.
- **7 agents**: 3 workflow (researcher, scorer, validator), 1 combined role agent (`pom-trio`), 3 audit (discovery, portfolio, ethics).
- **4 orchestrator commands**: `/pom`, `/pom-funnel`, `/pom-new-product`, `/pom-review`.
- **Bundled methodology** under `methodology/`: references, workflows, templates, generic industry baseline.

## Install

```text
/plugin marketplace add swansoncreativestudios/product-operating-model
/plugin install pom-core@product-operating-model
```
